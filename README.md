# PaqueExpress

Aplicación para gestionar paquetes con **API en FastAPI** y **App Flutter**.

## Requisitos
- MySQL
- Python 3.10+
- Flutter SDK
- Paquetes Python: `fastapi`, `uvicorn`, `mysql-connector-python`
- Paquetes Flutter: `http`

## Instalación

### Base de datos
1. Abrir MySQL y ejecutar `database.sql` para crear la base de datos y tablas.

### API
1. Instalar dependencias:
   ```bash
   pip install fastapi uvicorn mysql-connector-python
